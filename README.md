# Hue Hopper
#### Version 0.3.0

![](https://github.com/sugarvoid/hue-hopper/blob/Current/docs/hue.gif)

## Description
A game about rotating and bouncing. First game I made so the code isn't as clean/structured as I'd like. Still a work in progress. 

## Controls 
- **Move** [A] and [D]
- **Rotate** Arrown Keys
- **Slam** Space
- **Pause** Esc
