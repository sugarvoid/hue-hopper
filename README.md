## Description
A game about rotating and bouncing. This is the first "game" I've made so the code isn't as clean/structured as I'd like. Still slowly working on refactoring as I learn better practices.

## Controls 
- **Move** [A] and [D]
- **Rotate** Arrown Keys
- **Slam** Space
- **Pause** Esc
