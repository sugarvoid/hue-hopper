# Hue Hopper
#### Version 0.3.0

![](https://github.com/sugarvoid/hue-hopper/blob/Current/docs/hue.gif)

## Description
A game about rotating and bouncing. 

## Controls 
**Move** 
- [A] and [D]
**Rotate** 
- Arrown Keys
**Slam** 
- Space
**Pause** 
- Esc
